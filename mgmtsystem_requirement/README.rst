=================================
Management System - Requirement
=================================

This module enables you to add  Requirements of your management systems:

* Quality (ISO 9001)
* Environment (ISO 14001)
* Information Security (ISO 27001)
* Health and Safety (ISO 45001)
* IT Services (ISO 20000)

Installation
============

No installation required.
